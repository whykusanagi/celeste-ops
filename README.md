# CelesteOps

CelesteOps is a local-first creator control panel (macOS desktop app) for tasks, content pipeline, stream calendar, milestones, and documents — with an MCP layer so AI agents (Claude Code, Claude Desktop, Cursor, Codex, Celeste CLI) can drive it.

This repository is the **public distribution + agent kit**:

- **Releases** — the signed macOS app (`.dmg` / `.app.zip`) with PGP-signed checksums. See [VERIFY.md](./VERIFY.md).
- **The MCP shim + installer** — so any MCP client can talk to the running app.
- **Docs** — tool reference and connection/verification guides.

> The application source is maintained privately; this repo ships the built app and everything an agent needs to install and use it.
